# XWZX News

Vue 3 + Vite mobile-style news frontend.

## Features

- Home, category, detail, favorite, history, profile, login, and register pages
- Pinia state management
- Vue Router
- i18n language files
- Vant UI dependency
- AI chat page

## Run

```bash
npm install
npm run dev
```

Configure the backend API base URL in `.env` if needed.
